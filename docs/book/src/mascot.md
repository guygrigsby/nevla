# Nevli the mongoose

<img src="nevli.png" alt="Nevli the mongoose" width="280">

Nevli is the mascot; nevla is the language. Both are the mongoose in
Hindi: नेवला (nevlā) is the noun, and नेवली (nevlī) is a female
mongoose, which she is. The language borrows her disposition: small,
fast, and fatal to snakes.

The project was briefly named rikki;
[ADR 0014](https://github.com/guygrigsby/nevla/blob/main/docs/adr/0014-rename-to-nevla.md)
records the rename.

The artwork lives in
[`art/`](https://github.com/guygrigsby/nevla/tree/main/art) (CC BY 4.0;
the code is MIT and stays that way).
