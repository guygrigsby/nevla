import py "torch"

fn main() (error?) {
    w := check torch.randn([784, 10], requires_grad: true)
    x := check torch.randn([32, 784])
    logits := check (x @ w)
    print(check str(logits.shape))
    return none
}

That is real torch, one import away, with Python's exceptions arriving as typed error values instead of tracebacks. The whole program is checked before any of it runs; a checked program cannot crash the process.

Errors are values and dropping one is a compile error. There is no nil. Dependencies are declared or the program does not compile. The twenty-minute crash dies at nevla check, in milliseconds, or not at all.
